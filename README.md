# CodePilot Agent

CodePilot Agent is a local AI coding assistant extension for the Anti-Gravity IDE (VS Code). It integrates with LM Studio to provide private, high-performance coding support.

## Features
- **Local LLM Integration**: Uses LM Studio's API for inference.
- **Context Awareness**: Automatically pulls code from the active editor.
- **Security**: Filters malicious inputs and validates requests.
- **Premium UI**: Modern dark mode with glassmorphism.

## Getting Started
1. Run `npm install` to install dependencies.
2. Run `npm run compile` to build the extension.
3. Open LM Studio and start the local server on port 1234.
4. Press `F5` in VS Code to test.

## Requirements
- VS Code 1.80.0+
- LM Studio (Running local server)
